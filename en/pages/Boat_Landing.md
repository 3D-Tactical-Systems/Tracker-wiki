# Boat Landing

The 3D Tactical System tracker is designed to be used in harsh and demanding enviroments. Besides its rugged build it has functionality that allows it to be used on a moving or floating platforms. This includes the following:

* Accurate heading alignment
* Integration into Ground Control Software (Mission Planner, QGroundControl)
